Intercooler 0.2.0 → 0.3.x Upgrade Guide

Fresh on the heels of saying that we were adopting Semantic Versioning in 0.2.0, we promptly violated Semantic Versioning.

Last time, promise! ☺

Upgrade Steps

Here are steps to upgrade from Intercooler 0.2.0 to 0.3.0:

  1. Rename the ic-load-on attribute to ic-trigger-on.
  2. All intercooler events that were previously named ic.* should be reversed to *.ic - this is more consistent with jQuery event name spacing.
  3. All logging related methods, setLogger, log, setLogLevel and logLevels were removed from the API and should be replaced by jQuery event listeners for the log.ic event.

Not too bad, but still, sorry about that!
